<a name="compare-two-doubles"></a>
>Сравнение двух чисел типа double

Введите 2 числа и сравните их.\
Числа могут быть равны или приблизительно равны с точностью 0.01.

Особенности:
- проверка на ввод целых чисел и чисел с плавающей точкой
- зацикливание некорректного ввода с выходом по символу `|`
- использование функций
<br/><br/>

<a name="compare-few-doubles"></a>
>Интерактивное сравнение нескольких double

Введите поочередно 5 чисел и для каждого числа единицу измерения.\
По мере ввода число проверяется на максимальное, минимальное\
с учетом конвертации в общую единицу измерения (метры)\
и данная информация интерактивно обновляется.\
По завершению цикла выводятся min / max значения,\
количество, сумма чисел и сами числа в порядке возрастания.

Особенности:
- проверка на ввод целых чисел и чисел с плавающей точкой
- проверка единиц измерения на корректность
- использование динамического массива `vector`
<br/><br/>

Особенности:
- компактный алгоритм с циклом `while`
